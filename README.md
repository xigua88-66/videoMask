# 🎬 视频标注工具

一款专业的视频处理与图像标注软件，支持视频抽帧、图像标注、视频合成等功能。

## ✨ 主要功能

- 🎬 **视频抽帧**: 将视频按指定间隔提取为图片序列
- ✏️ **图像标注**: 支持矩形、多边形、文本标注
- 🎨 **标注编辑**: 颜色、粗细、字体等属性自定义
- 💾 **数据保存**: JSON格式保存标注元数据
- 🔄 **视频合成**: 将标注后的图片重新合成为视频
- 🗑️ **文件管理**: 支持任务和图片的删除管理

## 🚀 快速开始

### 安装
1. **macOS**: 双击 `videoMask.app` 应用程序
2. **Windows**: 双击 `videoMask.exe` 可执行文件
3. **源码运行**: `python app/main.py`

### 使用流程
1. 创建新任务
2. 上传视频并设置抽帧间隔
3. 开始抽帧处理
4. 进入图片列表，选择图片进行标注
5. 使用各种标注工具绘制标注
6. 保存标注数据
7. 合成最终视频

## 📁 项目结构

```
videoMask/
├── app/
│   ├── main.py              # 主应用程序
│   └── core/
│       └── video_processor.py  # 视频处理模块
├── data/                    # 数据目录
│   ├── tasks.json          # 任务列表
│   └── 任务名/
│       ├── original_frames/    # 原始抽帧图片
│       └── annotated_frames/   # 标注后图片和JSON数据
├── build/                   # 编译输出目录
├── dist/                    # 发布目录
├── requirements.txt         # Python依赖
├── videoMask.spec          # macOS编译配置
├── videoMask_windows.spec  # Windows编译配置
└── README.md               # 项目说明
```

## 🐛 问题排查

### 日志文件
- **macOS**: `~/videoMask_logs/`
- **Windows**: `%USERPROFILE%\videoMask_logs\`

### 常见问题
1. **应用程序无法启动**: 检查系统权限和兼容性
2. **视频抽帧失败**: 检查视频格式和文件权限
3. **标注不显示**: 检查标注数据和图片文件
4. **视频合成失败**: 检查标注图片和输出路径

详细解决方案请参考 [故障排除指南](故障排除指南.md)
