# 📖 视频标注工具使用手册

## 🎯 软件概述

**视频标注工具**是一款专业的视频处理与图像标注软件，支持视频抽帧、图像标注、视频合成等功能。适用于视频分析、数据标注、教学演示等场景。

### 主要功能
- 🎬 **视频抽帧**：将视频按指定间隔提取为图片序列
- ✏️ **图像标注**：支持矩形、多边形、文本标注
- 🎨 **标注编辑**：颜色、粗细、字体等属性自定义
- 💾 **数据保存**：JSON格式保存标注元数据
- 🔄 **视频合成**：将标注后的图片重新合成为视频
- 🗑️ **文件管理**：支持任务和图片的删除管理

## 🚀 安装与启动

### 系统要求
- **macOS**: 10.14 或更高版本
- **Windows**: Windows 10 或更高版本
- **内存**: 建议 4GB 以上
- **存储**: 至少 1GB 可用空间

### 安装方式
1. **macOS**: 双击 `videoMask.app` 应用程序
2. **Windows**: 双击 `videoMask.exe` 可执行文件
3. **源码运行**: `python app/main.py`

### 启动说明
- 首次启动可能需要授权文件访问权限
- 如果应用程序无法启动，请尝试运行可执行文件版本
- 日志文件保存在：`~/videoMask_logs/` 目录

---

## 📋 任务管理

### 创建新任务
1. 启动软件后，在主界面点击"创建新任务"
2. 输入任务名称（建议使用有意义的名称）
3. 点击"确定"创建任务

### 任务列表功能
- **📁 任务状态**: 显示任务是否已合成视频
  - `📁`: 未合成视频
  - `🎬`: 已合成视频
- **删除任务**: 选中任务后点击"删除选中任务"按钮
- **打开任务**: 双击任务名称进入图片列表

### 任务文件结构
```
data/
├── 任务名/
│   ├── original_frames/     # 原始抽帧图片
│   └── annotated_frames/    # 标注后图片和JSON数据
└── tasks.json              # 任务列表数据
```

## 🎬 视频抽帧功能

### 操作步骤
1. 在任务列表中点击"创建新任务"
2. 输入任务名称
3. 点击"上传视频"选择视频文件
4. 设置抽帧间隔（秒）
   - 建议值：1-5秒
   - 间隔越小，图片越多，处理时间越长
5. 点击"开始抽帧"
6. 等待处理完成

### 支持的视频格式
- **MP4** (推荐)
- **AVI**
- **MOV**
- **MKV**
- **WMV**

### 抽帧结果
- 原始图片保存在：`data/任务名/original_frames/`
- 图片命名格式：`frame_000000.png`, `frame_000001.png`...
- 图片按视频帧顺序排列

### 常见问题
- **抽帧失败**: 检查视频格式是否支持
- **处理时间过长**: 减少抽帧间隔或使用更短的视频
- **内存不足**: 关闭其他应用程序释放内存

## ✏️ 图像标注功能

### 进入标注界面
1. 在任务列表中双击任务名称
2. 在图片列表中双击要标注的图片
3. 进入标注工作区

### 标注工具说明

#### 矩形标注
- 点击工具栏"矩形"按钮
- 在图片上拖拽绘制矩形框
- 支持移动和调整大小
- 单击选中，双击编辑属性

#### 多边形标注
- 点击工具栏"多边形"按钮
- 在图片上点击多个点绘制多边形
- 按回车键完成绘制
- 单击选中，双击编辑属性

#### 文本标注
- 点击工具栏"文本"按钮
- 在图片上点击位置
- 在弹出的对话框中输入文字
- 单击选中，双击编辑属性

### 标注属性设置

#### 颜色设置
- 点击工具栏"颜色"按钮
- 在颜色选择器中选择颜色
- 支持所有标准颜色
- 实时预览效果

#### 粗细设置
- 使用工具栏粗细滑块
- 范围：1-20像素
- 实时预览效果

#### 字体设置
- 字号：8-72像素
- 粗体：点击"B"按钮切换
- 实时调整预览

### 标注编辑功能

#### 选择标注
- **单击**: 进入移动模式（蓝色虚线框）
- **双击**: 进入编辑模式（绿色虚线框）

#### 移动标注
- 选中后直接拖拽移动
- 实时显示移动轨迹
- 支持精确位置调整

#### 编辑属性
- 双击进入编辑模式
- 修改颜色、粗细、字号等属性
- 按回车键确认修改
- 按ESC键取消编辑

#### 删除标注
- 选中标注后按Delete键
- 或使用工具栏"撤销"按钮

## 💾 标注保存与预览

### 保存标注
1. 完成标注后点击工具栏"保存"按钮
2. 标注数据保存到JSON文件
3. 标注图片保存到annotated_frames目录
4. 保存后可以继续编辑

### 预览效果
1. 保存标注后点击"🔍 预览效果"按钮
2. 弹窗显示最终的标注图片
3. 关闭预览返回编辑状态

### 重置标注
1. 点击工具栏"重置"按钮
2. 清空所有标注数据
3. 恢复原始图片状态

### 数据格式
```json
{
  "annotations": [
    {
      "type": "rectangle",
      "points": [[x1, y1], [x2, y2]],
      "color": "#ff0000",
      "thickness": 2
    },
    {
      "type": "text",
      "pos": [x, y],
      "text": "标注文本",
      "color": "#0000ff",
      "font_size": 16,
      "bold": false
    }
  ]
}
```

## 🔄 视频合成功能

### 合成步骤
1. 完成所有图片的标注
2. 在任务列表中点击"合成视频"按钮
3. 选择保存路径和文件名
4. 等待合成完成

### 合成设置
- **输出格式**: MP4
- **帧率**: 30fps（默认）
- **质量**: 保持原始质量
- **编码**: H.264

### 合成结果
- 视频文件保存在项目根目录
- 命名格式：`任务名_annotated.mp4`
- 包含所有标注信息

### 性能优化
- 合成前确保所有图片都已标注
- 大视频建议分批处理
- 合成过程中避免其他操作

## 🗑️ 文件管理

### 图片管理
- **查看状态**: 图片列表显示标注状态
  - `⚪`: 未标注
  - `✅`: 已标注
- **删除图片**: 选中图片后点击"删除选中图片"
- **批量操作**: 支持多选删除

### 任务管理
- **删除任务**: 删除整个任务及其所有文件
- **任务状态**: 实时显示视频合成状态
- **数据备份**: 建议定期备份data目录

### 删除功能
- **删除当前图片**: 在标注界面点击"🗑️ 删除当前图片"
- **自动切换**: 删除后自动切换到下一张图片
- **安全确认**: 删除前会弹出确认对话框

## 🔧 常见问题排查

### 1. 应用程序崩溃问题

#### 问题现象
- 应用程序突然退出
- 无法正常启动
- 功能异常

#### 排查步骤
1. **检查日志**
   ```bash
   # macOS
   ls ~/videoMask_logs/
   cat ~/videoMask_logs/videoMask_*.log
   
   # 查看系统日志
   log show --predicate 'process == "videoMask"' --last 1h
   ```

2. **检查权限**
   - 确保应用程序有文件访问权限
   - 检查视频文件是否可读

3. **检查系统兼容性**
   - 确认系统版本符合要求
   - 检查是否有必要的系统库

#### 解决方案
- 重新安装应用程序
- 检查系统更新
- 使用可执行文件版本
- 查看日志文件获取详细错误信息

### 2. 视频抽帧问题

#### 问题现象
- 抽帧失败
- 图片质量差
- 处理时间过长

#### 排查步骤
1. **检查视频格式**
   - 确认视频格式被支持
   - 检查视频文件是否损坏

2. **检查抽帧参数**
   - 调整抽帧间隔
   - 检查视频时长

3. **检查存储空间**
   - 确保有足够的磁盘空间
   - 检查写入权限

#### 解决方案
- 转换视频格式为MP4
- 减少抽帧间隔
- 清理磁盘空间
- 使用更短的视频进行测试

### 3. 标注显示问题

#### 问题现象
- 标注不显示
- 标注位置偏移
- 标注属性异常

#### 排查步骤
1. **检查标注数据**
   - 查看JSON文件是否正常
   - 检查坐标数据

2. **检查图片文件**
   - 确认图片文件存在
   - 检查图片格式

3. **重置标注**
   - 使用重置功能清空标注
   - 重新进行标注

#### 解决方案
- 删除对应的JSON文件重新标注
- 检查图片文件完整性
- 重新启动应用程序
- 使用预览功能检查标注效果

### 4. 视频合成问题

#### 问题现象
- 合成失败
- 视频质量差
- 合成时间过长

#### 排查步骤
1. **检查标注图片**
   - 确认所有图片都已标注
   - 检查图片文件完整性

2. **检查输出路径**
   - 确认输出路径可写
   - 检查磁盘空间

3. **检查合成参数**
   - 调整帧率设置
   - 检查图片分辨率

#### 解决方案
- 重新标注缺失的图片
- 更换输出路径
- 调整合成参数
- 检查系统资源使用情况

### 5. 性能优化建议

#### 硬件要求
- **CPU**: 建议4核以上
- **内存**: 建议8GB以上
- **存储**: 建议SSD硬盘

#### 软件优化
- 定期清理临时文件
- 关闭不必要的后台程序
- 使用最新版本软件

#### 使用技巧
- 合理设置抽帧间隔
- 及时保存标注数据
- 定期备份重要数据
- 使用预览功能检查效果
